## Зачем этот документ

Стандарт фиксирует новую норму производительности: сотрудник должен не просто выполнять работу вручную, а выбирать наиболее эффективный способ — самостоятельно, с помощью ИИ, цифрового агента, шаблона или автоматизации.

Документ отвечает на 7 вопросов:

1. **Зачем он нужен?** — зафиксировать новую норму производительности
2. **Для кого обязателен?** — для всех, с разной глубиной по уровням ролей
3. **Что считается нормой?** — работа через ИИ, агентов, шаблоны, автоматизацию и проверку качества
4. **Какие уровни зрелости?** — от AI User (A1) до AI Transformation Leader (A6)
5. **Что требуется от ролей?** — ТОПы, руководители, лиды, эксперты, специалисты
6. **Как применяется?** — найм, адаптация, обучение, оценка, грейдинг, кадровый резерв
7. **Как проверяется?** — через поведение, кейсы, результаты, автоматизированные сценарии

## Шкала зрелости (краткая)

| Уровень | Название | Главный признак |
|---|---|---|
| **A1** | AI User | Использует ИИ для простых личных задач |
| **A2** | Agent Operator | Умеет ставить рабочие задачи ИИ/агенту |
| **A3** | Workflow Builder | Создаёт повторяемые AI-сценарии и шаблоны |
| **A4** | Agent Supervisor | Управляет агентами и качеством их работы в команде |
| **A5** | Agentic Process Owner | Перестраивает процессы вокруг ИИ и агентов |
| **A6** | AI Transformation Leader | Меняет модель работы функции или компании |

Подробнее — [раздел 5](05-maturity-scale/05-maturity-scale.md).

## Главный принцип

> **Не делай вручную то, что можно безопасно поручить ИИ, цифровому агенту, шаблону или автоматизированному сценарию.**

При этом ИИ и цифровые агенты не снимают ответственность с сотрудника. Сотрудник отвечает за постановку задачи, качество результата, проверку выводов, соблюдение правил безопасности и применение результата в реальной работе.
