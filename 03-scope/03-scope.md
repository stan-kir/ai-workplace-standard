# 3. Область применения

Стандарт обязателен для всех сотрудников компании, но глубина требований зависит от уровня роли, зоны ответственности и степени влияния на процессы.

Стандарт применяется в следующих областях:

| Область                | Как применяется                                                                                      |
| ---------------------- | ---------------------------------------------------------------------------------------------------- |
| **Найм**               | Используется как фильтр готовности сотрудника работать в новой производительной среде                |
| **Адаптация**          | Новый сотрудник осваивает базовые AI-инструменты, правила безопасности и рабочие сценарии            |
| **Обучение**           | По шкале стандарта строятся программы развития сотрудников и руководителей                           |
| **Оценка**             | Проверяется не факт использования ИИ, а реальное применение для скорости, качества и результата      |
| **Грейдинг**           | Уровень AI/agent-first зрелости включается в требования к ролям                                      |
| **Карьерный рост**     | Рост в роли требует соответствующего уровня работы с ИИ, агентами и автоматизацией                   |
| **Performance review** | Оценивается вклад сотрудника в снятие рутины, ускорение процессов и повышение качества               |
| **Кадровый резерв**    | Сотрудники с высокой AI-зрелостью получают преимущество при развитии в лиды и руководители           |
| **Оргдизайн**          | Помогает пересматривать роли, процессы и зоны ответственности                                        |
| **Процессы**           | Повторяемые операции проверяются на возможность передачи ИИ, агентам, шаблонам или автоматизации     |
| **Безопасность**       | Фиксирует правила работы с данными, документами, доступами и результатами ИИ                         |
| **Культура**           | Формирует норму самостоятельности: сначала попробуй решить через ИИ/агента/сценарий, потом эскалируй |
